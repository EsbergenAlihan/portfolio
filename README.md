# 🤖 Портфолио «Обо мне» — Telegram-бот

Telegram-бот-портфолио для конкурса CapEducation (курс Middle Python).

## 🎯 Что умеет бот

- 8 разделов портфолио через команды и inline-кнопки
- Удобная навигация «Назад в меню»
- Обработка свободного текста с помощью регулярных выражений (re)
- Аргументы командной строки (sys.argv) для токена и debug-режима
- Контент вынесен в отдельный файл (content.py)

## 📋 Команды

| Команда | Раздел |
|---------|--------|
| `/start`, `/menu` | Главное меню |
| `/about` | О себе |
| `/goal` | Моя цель |
| `/story` | Как я пришёл в IT |
| `/mentor` | Мой ментор |
| `/progress` | Точка А → Точка Б |
| `/hobbies` | Хобби и интересы |
| `/works` | Мои лучшие работы |
| `/github` | Ссылка на GitHub |
